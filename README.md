# Discover Sri Lanka – Step 4 Webpage Project

This repository contains my **CodePen webpage project** about Sri Lanka – “The Pearl of the Indian Ocean.”  
It includes images, text, and a popup map feature.  

---

## Features

- **Introduction** about Sri Lanka  
- **Centered main image**  
- Multiple **images of landscapes, beaches, and mountains**  
- **Popup map** showing Sri Lanka  
- **Ordered and unordered lists** for top places to visit  
- Fully written **in HTML, CSS, and JS**  

---

## Images Used

1. Sigiriya Rock Fortress  
2. Mountains and scenic views  
3. Oceans and beaches  
4. Map of Sri Lanka (in popup)

## About the Project

This project was created as part of the **ITD Step 4 course**.  
It demonstrates:

- CSS positioning (`relative`, `absolute`, `fixed`)  
- Centering images  
- Z-index usage  
- Flexbox layout  
- Popup modal functionality
